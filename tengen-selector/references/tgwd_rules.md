# TGWD系列万能式断路器选型规则

## 型号命名规则

TGWD-[壳架]/[极数] [额定电流] [安装方式] [控制器类型] [控制电压] [附件]

### 基础参数

| 参数 | 可选值 |
|------|--------|
| 壳架 | 1000, 2000, 3200, 4000, 6300 |
| 极数 | 3P, 4P |
| 额定电流 | 根据壳架确定 |
| 安装方式 | 固定水平、抽屉水平 |
| 控制器 | 2H型、3H型、3M型 |
| 控制电压 | AC220V, DC220V |

### 与TGW1N的区别

| 特性 | TGWD | TGW1N |
|------|------|-------|
| 控制器 | 2H型/3H型/3M型 | 祥云3.0（M型）/3H型 |
| 安装方式 | 固定水平、抽屉水平 | 固定式、抽屉水平、抽屉垂直 |
| 附件标识 | 欠压延时Xs、带四遥功能 | 祥云3.0标识 |
| 接线端子 | 55回路、长排/短排 | 56回路 |

### 特殊标识

- **(装锁)** - 带锁功能
- **(U)** - 特殊版本
- **(长排)** - 长排接线
- **(短排)** - 短排接线
- **(FJJP)** - 福建精品
- **Ir1=0.8In** - 长延时整定值
- **欠压延时Xs** - 欠压脱扣延时时间

### 附件类型

| 附件 | 说明 |
|------|------|
| 欠压延时Xs | 欠压脱扣器带延时 |
| 带四遥功能 | 遥控、遥信、遥测、遥调 |
| 独立四开四闭辅助 | 辅助触头配置 |
| 55回路接线端子 | 接线端子数量 |
| 六组转换触头 | 转换触头配置 |
| 外接零序互感器 | 漏电保护 |

## 常用型号示例

### 基础款
```
TGWD-2000/3P 1600A 抽屉水平 3M型 DC220V
```

### 带欠压
```
TGWD-2000/3P 800A 固定水平 AC220V 欠压延时5s
```

### 带四遥
```
TGWD-3200/3P 2000A 抽屉水平 3H型 AC220V 带四遥功能(长排)
```

### 带漏电
```
TGWD-1000/4P 1000A 抽屉水平 3H型 AC220V 欠压延3s 带漏电保护功能 外接零序互感器 漏电设置值500mA
```

## EAP3查询关键词

- 带附件：`TGWD-2000 欠压`、`TGWD-2000 四遥`
- 查控制器：`TGWD-2000 3H型`、`TGWD-2000 2H型`
