# TGW1N系列万能式断路器选型规则

## 型号命名规则

TGW1N-[壳架]/[极数] [额定电流] [安装方式] [控制器类型] [控制电压] [附件]

### 基础参数

| 参数 | 可选值 |
|------|--------|
| 壳架 | 1000, 1600, 2000, 2500, 3200, 4000, 6300 |
| 极数 | 3P, 4P |
| 额定电流 | 根据壳架确定（如1000A壳架可选400A-1000A） |
| 安装方式 | 固定式、抽屉水平、抽屉垂直 |
| 控制器 | 祥云3.0（M型标配）、3H型 |
| 控制电压 | AC220V, AC230V, AC380V, DC220V |

### 附件类型

| 附件 | 说明 |
|------|------|
| 欠压 | 欠压脱扣器 |
| 漏电 | 外接零序互感器（3P漏电标配） |
| 检有压 | 两种形式：外挂模块（便宜）或3H型内置（贵） |
| 分励 | 分励脱扣器 |
| 辅助触头 | 转换触头组 |

## 选型流程

1. **确认基础参数** → 壳架、极数、安装方式
2. **选择附件** → 欠压、漏电、检有压等
3. **检有压形式选择**（如需要）：
   - 外挂检有压模块（便宜，不限制控制器）
   - 带检有压合闸功能（必须3H控制器，贵）
4. **系统检索** → 查EAP3是否有完全匹配型号
5. **方案选择**：
   - A: 系统现有3H型（贵但有物料）
   - B: 祥云3.0+外挂模块（便宜走定制）
6. **输出完整型号和物料编码**

## 常用型号示例

### 基础款（无附件）
```
TGW1N-2000/3P 2000A 抽屉水平 AC220V（祥云3.0）
```

### 带漏电
```
TGW1N-2000/3P 2000A 抽屉水平 AC220V 带漏电保护功能 外接零序互感器（祥云3.0）
```

### 带检有压（外挂模块方案-便宜）
```
TGW1N-2000/3P 2000A 抽屉水平 AC220V 外挂检有压合闸模块（祥云3.0）
```

### 带检有压（3H型方案-贵）
```
TGW1N-2000/3P 2000A 抽屉水平 3H型 AC220V 带检有压合闸功能（祥云3.0）
```

## EAP3查询关键词

- 基础查询：`TGW1N-2000/3P 2000A 抽屉`
- 带附件：`TGW1N-2000 漏电`、`TGW1N-2000 欠压`
- 查3H型：`TGW1N-2000H`
