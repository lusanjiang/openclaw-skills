# 天正电气断路器选型助手

用于TGW1N和TGWD系列万能式断路器的智能选型与价格查询。

## 快速调用标签

| 标签 | 用途 | 示例 |
|------|------|------|
| `#tengen` | 天正电气全系列产品 | #tengen TGM1-125 |
| `#tgw` | TGW/TGWD/TGW1N 框架断路器选型 | #tgw 2000/4P 1250A 固定式 |

**注意**: 使用 `#tgw` 标签时，系统会自动调用 EAP3 查询相关物料的价格和货期。

## 核心功能

1. **智能选型** - 根据需求参数生成推荐型号
2. **价格查询** - 连接EAP3系统查询实时价格和货期
3. **方案对比** - 系统现有 vs 定制方案对比

---

## 通用选型操作流程（适用于TGW1N/TGWD/TGM1N）

### 标准选型流程

```
需求分析 → 参数匹配 → 型号生成 → EAP3查询 → 结果输出
```

### 详细步骤

| 步骤 | 操作 | 说明 |
|------|------|------|
| **1. 需求分析** | 解析用户输入 | 提取电流、分断能力、极数、附件、电压等参数 |
| **2. 壳架选择** | 匹配壳架等级 | 根据电流选择：63/125/160/250/320/400/630/800/1250/1600/2000/3200/4000 |
| **3. 分段确定** | 确定分断能力 | L(25kA)/M(35-50kA)/H(50-70kA)/R(70-85kA) |
| **4. 极数选择** | 确定极数 | 2P/3P/3N/4P，注意400A以上无2P |
| **5. 附件配置** | 选择附件代码 | 分励/欠压/辅助/报警等组合 |
| **6. 电压标注** | 确认控制电压 | AC220V/AC380V/DC24V |
| **7. EAP3查询** | 查询价格和货期 | 调用eap3_client.py查询 |
| **8. 输出结果** | 生成完整型号 | 基础型号 + 电流 + 电压 + 备注 |

### 输出规范

**必须包含**：
- 基础型号（系列+壳架+分段+附件）
- 额定电流
- 控制电压
- 备注标识

**格式**：
```
[系列]-[壳架][分段]/[极数][脱扣][附件] [电流] [电压](祥云3.0)
```

**示例**：
- TGM1N: `TGM1N-160M/3370 160A AC220V(祥云3.0)`
- TGW1N: `TGW1N-2000/3P 2000A AC220V(祥云3.0)`

## 通用选型规则（适用于TGW1N/TGWD/TGM1N）

### 完整型号输出规范
```
[系列]-[壳架][分段]/[极数][脱扣][附件][安装方式] [电流] [电压] [备注]
```

### 必须包含的要素
- **基础型号**: 系列+壳架+分段+附件代码
- **电流**: 额定电流（如160A）
- **电压**: 控制电压（如AC220V/DC24V）
- **备注**: (祥云3.0)/(标准款)/(大体积)等

### 输出示例
| 系列 | 完整型号 |
|------|----------|
| TGW1N | `TGW1N-2000/3P 2000A AC220V(祥云3.0)` |
| TGM1N | `TGM1N-160M/3370 160A AC220V(祥云3.0)` |

### 禁止简化
❌ 错误：`TGM1N-160M/3370` 或 `TGW1N-2000/3P`  
✅ 正确：`TGM1N-160M/3370 160A AC220V(祥云3.0)`  
✅ 正确：`TGW1N-2000/3P 2000A AC220V(祥云3.0)`

## 选型流程

1. 收集基础参数（壳架、极数、安装方式）
2. 确认附件需求（欠压、漏电、检有压等）
3. 检有压形式选择（外挂模块便宜 vs 3H型贵）
4. EAP3系统查询价格
5. 输出推荐方案

## 使用脚本

价格查询脚本位于 `scripts/eap3_client.py`：

```python
from scripts.eap3_client import query_tgw1n_price

# 查询价格
results = query_tgw1n_price("TGW1N-2000/3P 2000A 抽屉")

for item in results[:5]:
    print(f"物料编码: {item['物料编码']}")
    print(f"型号: {item['型号']}")
    print(f"面价: {item['面价']}")
    print(f"货期: {item['货期']}")
```

命令行使用：
```bash
python3 scripts/eap3_client.py "TGW1N-2000/3P 2000A 抽屉"
```
