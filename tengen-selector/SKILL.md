## 🏗️ 工作流架构 (v2.0 新特性)

学习自 Coze 工作流设计理念，选型流程可视化编排：

```
用户输入 → 解析参数 → 匹配壳架 → 确定分断 → 生成型号 → EAP3查询 → 输出结果
    │           │           │           │           │           │
    ▼           ▼           ▼           ▼           ▼           ▼
  #tgw      提取电流    向上取整     L/M/H      TGW1N-    价格货期   格式化
            极数等     选壳架等级   分断代码    2000/3P              报告
```

### 节点类型
| 节点 | 功能 | 示例 |
|:---|:---|:---|
| `parse_input` | 解析用户输入 | 提取"TGW1N 2000A 3P"→系列/电流/极数 |
| `match_frame` | 匹配壳架等级 | 2000A→匹配2000壳架 |
| `determine_breaking` | 确定分断能力 | 默认M型(35-50kA) |
| `generate_model` | 生成基础型号 | TGW1N-2000/3P |
| `query_eap3` | EAP3价格查询 | 调用API获取价格货期 |
| `format_output` | 格式化输出 | 生成可读报告 |

## 选型流程

### 标准选型工作流

| 步骤 | 节点 | 说明 |
|------|------|------|
| **1** | `parse_input` | 解析用户输入，提取系列、电流、极数、安装方式 |
| **2** | `match_frame` | 根据电流匹配壳架等级（向上取整）|
| **3** | `determine_breaking` | 确定分断能力代码 L/M/H/R |
| **4** | `generate_model` | 生成基础型号（如 TGW1N-2000/3P）|
| **5** | `query_eap3` | 调用EAP3查询价格和货期 |
| **6** | `format_output` | 格式化输出完整型号和建议 |

### 输出规范

**必须包含**：
- 基础型号（系列+壳架+分段+附件）
- 额定电流
- 控制电压
- 备注标识

**格式**：
```
[系列]-[壳架][分段]/[极数][脱扣][附件] [电流] [电压](控制器类型)
```

**示例**：
- TGM1N: `TGM1N-160M/3370 160A AC220V(祥云3.0)`
- TGW1N: `TGW1N-2000/3P 2000A AC220V(祥云3.0)`
- TGWD: `TGWD-2000/3P 2000A AC220V(3H型)`

### 壳架等级
| 电流需求 | 匹配壳架 |
|:---|:---|
| ≤63A | 63 |
| ≤125A | 125 |
| ≤160A | 160 |

| ≤400A | 400 |
| ≤630A | 630 |
| ≤800A | 800 |
| ≤1250A | 1250 |
| ≤1600A | 1600 |
| ≤2000A | 2000 |
| ≤3200A | 3200 |
| ≤4000A | 4000 |

### 分断能力代码
| 代码 | 分断能力 | 适用 |
|:---|:---|:---|
| L | 25kA | 低要求场合 |
| M | 35-50kA | **默认推荐** |
| H | 50-70kA | 高分断要求 |
| R | 70-85kA | 极高要求 |

### 极数选择
| 极数 | 适用场景 |
|:---|:---|
| 2P | 小电流单相 |
| 3P | 三相三线（最常见）|
| 4P | 三相四线（带N极）|

## 注意事项

1. EAP3系统需要登录，脚本已内置登录凭证
2. TGW1N/TGWD 最小壳架为2000A，小电流建议选用TGM1N系列
3. 检有压功能有两种实现方式（外挂模块 vs 3H型），价格差异大
4. 3P漏电保护需外接零序互感器
