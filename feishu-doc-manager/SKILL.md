## 三、更新规则

### 3.1 何时更新

**必须更新飞书文档的情况：**
- 本地 skill 文件有重大更新
- 新增关键知识点或经验
- 用户明确要求备份到飞书
- 项目里程碑完成

### 3.2 更新流程

1. **检查本地 skill 文件** - 确认最新内容
2. **读取飞书文档当前内容** - 避免重复或冲突
3. **分段追加内容** - 飞书 API 限制，分段写入更稳定
4. **验证更新成功** - 读取确认内容已写入
5. **提示用户** - 告知飞书文档已更新

### 3.3 注意事项

- **表格不支持**：用普通文本替代表格
- **分段写入**：一次性写入大量内容会失败
- **链接保留**：飞书文档链接永久有效，记录好文档ID
