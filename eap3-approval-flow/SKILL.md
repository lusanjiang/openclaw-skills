# EAP3 审批流程自动化 Skill

## 快速使用

使用标签 `#审批流程` 快速调用：

```
#审批流程
#审批流程 处理待办
```

## 功能

- 自动登录EAP3系统
- 打开待办审批页面
- 提取并显示定制物料详情
- 等待用户确认后自动提交审批
- 支持"已核实请后台支持"等审批选项
- **自动记录审批信息到飞书文档**

## 飞书文档记录

每次审批完成后，自动记录以下信息到飞书文档：
- 审批时间
- 单据编号（XZ38-XXXXXX）
- 申请人
- 客户名称
- 定制物料详情（型号规格）
- 审批结果
- 下一节点

**飞书文档链接：** https://feishu.cn/docx/Otlxd1AllogLYAxzPq4casnRnpb

## 审批流程

### 1. 自动处理步骤
1. 登录EAP3获取SID
2. 获取未读通知/待办列表
3. 识别用户的待办（福建/江西团队）
4. 打开待办详情页面
5. 提取定制物料信息
6. **弹出物料详情等待用户确认**
7. 用户回复"可以"后自动提交

### 2. 物料信息提取
- 定制系列（如：接触器类.接触器.CJX1）
- 生产公司（如：IO_温州工控二车间）
- 环保属性
- 定制描述（如：CJX1-32/2 32A DC36V）
- 参考价格

### 3. 审批选项
- 已提供面价解答
- **已核实请后台支持**（默认）
- 无效流程驳回

## 技术实现

- 调用API：`com.actionsoft.apps.notification_load_unread_msg`
- 审批接口：`CLIENT_BPM_FORM_MAIN_PAGE_OPEN`
- 提交操作：Playwright浏览器自动化

## 注意事项

- 只处理福建/江西团队的待办
- 需要用户确认后才提交（自动审批模式除外）
- 支持并行审批（多人同时处理）
- 审批后自动流转到下一节点
- **所有审批记录自动同步到飞书文档**
- 遇到错误自动重试并记录
