## 选型流程

1. **确定产品类别** → 根据应用场景选择大类
2. **确定技术参数** → 电流、电压、极数、安装方式
3. **选择具体系列** → 根据定位选择经济型/标准型/高端型
4. **配置附件** → 欠压、分励、辅助触头等
5. **查询价格货期** → EAP3系统实时查询

## 价格查询

使用EAP3客户端脚本查询实时价格：
```python
from scripts.eap3_client import query_tengen_price
results = query_tengen_price("TGM1-100/3P")
```
