# TGR系列热继电器

## 产品概述
TGR系列热继电器适用于交流50Hz，额定电压AC380V/AC660V，额定电流0.1A~630A的电路中，用于交流电动机的过载保护、断相保护及电流不平衡保护。

## 系列分类

### TGR1系列 - 标准型
- 额定电流：0.1A~95A
- 极数：3P
- 安装方式：与TGC1接触器配套安装
- 脱扣等级：10A级(标准)
- 适用场景：一般电机过载保护

### TGR2系列 - 大电流型
- 额定电流：25A~160A
- 极数：3P
- 安装方式：独立安装或与TGC2配套
- 适用场景：大功率电机保护

### TGR36系列 - 超大电流型
- 额定电流：160A~630A
- 极数：3P
- 安装方式：独立安装
- 适用场景：大电机、配电线路保护

## 型号命名规则

```
TGR[系列]-[额定电流范围]/[极数] [整定电流范围]
```

示例：
- `TGR1-0930/3P 0.63-1A` - TGR1系列，0.63-1A整定范围
- `TGR2-2536/3P 23-32A` - TGR2系列，23-32A整定范围

## 功能特点

1. **过载保护** - 双金属片热脱扣，反时限特性
2. **断相保护** - 三相不平衡或缺相时快速脱扣
3. **温度补偿** - 自动补偿环境温度影响
4. **手动/自动复位** - 可选择复位方式
5. **脱扣指示** - 脱扣后有明显指示

## 选型要点

1. **电流范围** - 电机额定电流应在整定范围的60%~100%之间
2. **脱扣等级** - 一般电机用10A级，重载启动用10级
3. **安装方式** - 与接触器配套或独立安装
4. **复位方式** - 根据现场需求选择手动或自动复位
