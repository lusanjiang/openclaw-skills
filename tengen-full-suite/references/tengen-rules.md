## 知识管理规则

### Skill 文件更新规则

**触发条件：**
- 获取到新的有关已保存 skill 的知识点
- 用户明确要求更新 skill
- 项目里程碑或重大进展

**更新流程：**
1. 更新本地 skill 文件
2. 同步更新相关飞书文档
3. **必须提示用户**："Skill 已更新，飞书文档已同步更新，请查看！"

### 飞书文档管理

**职责：** 使用 feishu-doc-manager skill 管理飞书文档

**规则：**
- 新建 skill 时，同时创建飞书文档备份
- 更新 skill 时，同步更新飞书文档
- 分段写入，避免 API 限制
- 记录文档ID和链接

### 用户通知义务

**必须通知用户的情况：**
- Skill 文件更新完成
- 飞书文档更新完成
- 新增重要知识点
- 项目里程碑达成

> **核心原则**：让用户及时了解所有更新和进展！
