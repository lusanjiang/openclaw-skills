# TGC系列交流接触器

## 产品概述
TGC系列交流接触器适用于交流50Hz，额定电压AC380V/AC660V，额定电流9A~800A的电路中，用于频繁接通和分断交流电动机及配电线路。

## 系列分类

### TGC1系列 - 标准型
- 额定电流：9A~95A
- 极数：3P, 4P
- 控制电压：AC24V/AC36V/AC48V/AC110V/AC220V/AC380V
- 使用类别：AC-3(一般电机)、AC-4(重载启动)
- 机械寿命：1000万次
- 电寿命：100万次(AC-3)

### TGC2系列 - 大电流型
- 额定电流：115A~800A
- 极数：3P, 4P
- 控制电压：AC110V/AC220V/AC380V/DC220V
- 适用场景：大功率电机控制、配电系统

## 型号命名规则

```
TGC[系列]-[额定电流]/[极数] [控制电压]
```

示例：
- `TGC1-0910/3P AC220V` - TGC1系列，9A，3P，AC220V控制
- `TGC2-185/3P AC380V` - TGC2系列，185A，3P，AC380V控制

## 选型要点

1. **额定电流** - 按电机额定电流的1.2~1.5倍选择
2. **使用类别** - 一般启动用AC-3，重载/反接制动用AC-4
3. **控制电压** - 与控制回路电压一致
4. **安装方式** - 导轨安装或螺丝固定

## 与热继电器的配合

接触器 + 热继电器 = 完整的电机保护方案
- TGC1系列配 TGR1系列热继
- TGC2系列配 TGR2/TGR36系列热继
