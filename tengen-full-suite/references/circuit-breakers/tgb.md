# TGB系列小型断路器

## 产品概述
TGB系列小型断路器适用于交流50Hz，额定电压AC230V/AC400V，额定电流1A~125A的终端配电网络中，用于保护线路和设备的过载、短路。

## 系列分类

### TGB1N系列 - 标准型
- 额定电流：1A~63A
- 分断能力：6kA
- 极数：1P, 2P, 3P, 4P, 1P+N, 3P+N
- 脱扣特性：B型(3-5In)、C型(5-10In)、D型(10-20In)
- 适用场景：家庭、商业、小型配电箱

### TGB1系列 - 经济型
- 额定电流：6A~63A
- 分断能力：4.5kA/6kA
- 极数：1P, 2P, 3P, 4P
- 脱扣特性：C型、D型
- 适用场景：普通照明、插座回路

### TGB2/TGB2N系列 - 高分断型
- 额定电流：1A~125A
- 分断能力：10kA
- 极数：1P~4P
- 适用场景：工业配电、高分断要求场合

## 型号命名规则

```
TGB[系列]-[极数] [额定电流] [脱扣特性] [附件]
```

示例：
- `TGB1N-63/2P C40` - TGB1N系列，2P，40A，C型脱扣
- `TGB2-125/3P D100` - TGB2系列，3P，100A，D型脱扣

## 脱扣特性选择

| 特性 | 脱扣范围 | 适用负载 |
|------|----------|----------|
| B型 | 3-5倍额定电流 | 阻性负载、照明 |
| C型 | 5-10倍额定电流 | 一般负载、插座 |
| D型 | 10-20倍额定电流 | 电机、变压器 |

## 附件配置

| 附件 | 说明 |
|------|------|
| 分励脱扣器 | 远程控制跳闸 |
| 欠压脱扣器 | 失压保护 |
| 辅助触头 | 状态指示 |
| 报警触头 | 故障报警 |
| 汇流排 | 多极并联安装 |

## 选型要点

1. **额定电流** - 按负载电流的1.25倍选择
2. **脱扣特性** - 照明用C型，电机用D型
3. **极数** - 单相用2P(1P+N)，三相用3P或4P
4. **分断能力** - 配电箱进线处选高分断(10kA)

## 与TGM/TGW的区别

| 对比项 | TGB(小型) | TGM(塑壳) | TGW(万能) |
|--------|-----------|-----------|-----------|
| 电流范围 | 1A~125A | 10A~1600A | 200A~6300A |
| 分断能力 | 4.5kA~10kA | 35kA~100kA | 50kA~100kA |
| 安装 | 导轨安装 | 螺丝固定 | 框架固定 |
| 适用 | 终端配电 | 分支回路 | 主进线 |
