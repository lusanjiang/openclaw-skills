# TGM系列塑壳断路器

## 产品概述
TGM系列塑料外壳式断路器适用于交流50Hz，额定电压AC400V/AC690V，额定电流10A~1600A的配电网络中，用于分配电能和保护线路及设备免受过载、短路等故障危害。

## 系列分类

### TGM1系列 - 经济型
- 额定电流：10A~800A
- 分断能力：35kA/50kA/65kA
- 极数：2P, 3P, 4P
- 安装方式：固定式、插入式
- 适用场景：普通配电、照明、小型动力

### TGM3系列 - 标准型
- 额定电流：16A~1600A
- 分断能力：50kA/65kA/80kA/100kA
- 极数：2P, 3P, 4P
- 安装方式：固定式、插入式、抽出式
- 适用场景：动力配电、电机保护、重要负荷

## 型号命名规则

```
TGM[系列]-[壳架]/[极数] [额定电流] [分断能力] [附件]
```

示例：
- `TGM1-100/3P 80A` - TGM1系列，100壳架，3P，80A
- `TGM3-250/3P 225A 高分断` - TGM3系列，250壳架，3P，225A，高分断型

## 选型要点

1. **壳架选择** - 根据额定电流选择，留20%余量
2. **分断能力** - 根据系统短路电流选择
3. **脱扣器类型** - 热磁式(标准)、单磁式(电机保护)、电子式(精准保护)
4. **安装方式** - 固定式(经济)、插入式(方便更换)、抽出式(不停电维护)
