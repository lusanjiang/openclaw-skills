# TGH系列隔离开关

## 产品概述
TGH系列隔离开关适用于交流50Hz，额定电压AC400V/AC690V，额定电流32A~3150A的配电系统中，用于不频繁接通和分断电路，以及隔离电源，确保检修安全。

## 系列分类

### TGH1系列 - 标准型
- 额定电流：32A~1600A
- 极数：3P, 4P
- 安装方式：柜内安装、导轨安装(小电流)
- 操作方式：柜内操作、柜外操作(加长手柄)
- 适用场景：配电柜、配电箱检修隔离

### TGH2系列 - 大电流型
- 额定电流：1600A~3150A
- 极数：3P, 4P
- 安装方式：柜内安装
- 适用场景：大电流配电系统

## 型号命名规则

```
TGH[系列]-[额定电流]/[极数] [安装方式] [操作方式]
```

示例：
- `TGH1-160/3P` - TGH1系列，160A，3P
- `TGH1-400/3P 柜外操作` - 400A，3P，柜外操作手柄

## 功能特点

1. **隔离功能** - 明显的断开点，确保检修安全
2. **不频繁操作** - 不适用于频繁通断场合
3. **无灭弧能力** - 不能带负荷分断
4. **机械联锁** - 可选配，防止误操作

## 与断路器的区别

| 对比项 | 隔离开关(TGH) | 断路器(TGM/TGW) |
|--------|---------------|-----------------|
| 保护功能 | 无 | 过载、短路保护 |
| 灭弧能力 | 无 | 有 |
| 操作频率 | 不频繁 | 可频繁操作 |
| 主要作用 | 隔离电源 | 保护电路 |
| 价格 | 低 | 高 |

## 选型要点

1. **额定电流** - 不小于回路最大工作电流
2. **极数** - 单相用2P，三相用3P或4P
3. **安装方式** - 根据柜体结构选择
4. **操作方式** - 柜内操作或柜外操作(安全考虑)

## 典型应用

1. **配电柜进线** - 主电源隔离
2. **配电柜出线** - 各回路隔离
3. **检修隔离** - 设备检修时断开电源
4. **双电源切换** - 与双电源开关配合使用

## 安装注意事项

1. 必须安装在无负荷状态下操作
2. 操作手柄应便于操作，有明显的分合指示
3. 大电流型号应考虑散热
4. 柜外操作时应确保操作手柄不会被误碰
