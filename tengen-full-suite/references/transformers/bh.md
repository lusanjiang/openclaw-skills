# BH系列电流互感器

## 产品概述
BH系列电流互感器适用于交流50Hz，额定电压AC0.66kV/AC0.5kV的配电系统中，用于电流测量、计量和保护。

## 系列分类

### BH-0.66系列 - 低压测量型
- 额定电压：AC660V
- 一次电流：5A~6000A
- 二次电流：5A/1A
- 精度等级：0.5级(测量)、0.5S级(计量)、10P10(保护)
- 安装方式：穿心式、母线式、开口式

### BH(SDH)系列 - 高精度型
- 额定电压：AC660V
- 精度等级：0.2级、0.2S级
- 适用场景：电能计量、精密测量

### BH-0.5系列 - 小型化
- 额定电压：AC500V
- 一次电流：5A~400A
- 适用场景：小型配电箱、空间受限场合

## 型号命名规则

```
BH-[电压等级] [结构尺寸] [一次电流]/[二次电流] [精度等级]
```

示例：
- `BH-0.66 60III 600/5 0.5级` - 0.66kV，60III型，600/5A，0.5级
- `BH(SDH)-0.66 100 2000/5 0.5级` - 高精度型，2000/5A

## 精度等级

| 等级 | 用途 | 误差限值 |
|------|------|----------|
| 0.2级 | 精密测量 | ±0.2% |
| 0.5级 | 一般测量 | ±0.5% |
| 1.0级 | 粗略测量 | ±1.0% |

| 0.5S级 | 计量(宽量程) | ±0.5%(1%~120%In) |
| 10P10 | 保护 | 复合误差≤10% |

## 选型要点

1. **额定电流** - 按回路最大电流选择，一般取1.2~1.5倍
2. **变比选择** - 常用变比：50/5、100/5、200/5、300/5、400/5、600/5、800/5、1000/5、1500/5、2000/5
3. **精度等级** - 计量用0.5S级，测量用0.5级，保护用10P10
4. **窗口尺寸** - 根据母排或电缆外径选择
5. **二次负荷** - 确保实际负荷≤额定负荷

## 安装方式

1. **穿心式** - 电缆或母排穿过互感器中心孔
2. **母线式** - 直接安装在母排上
3. **开口式** - 不停电安装，适合改造项目
