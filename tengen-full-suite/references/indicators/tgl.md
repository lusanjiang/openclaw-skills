# TGL系列按钮与指示灯

## 产品概述
TGL系列按钮开关和指示灯适用于交流50Hz，额定电压AC380V/AC220V，额定电流5A/10A的控制回路中，用于控制电路的接通、分断和状态指示。

## 系列分类

### TGLA系列 - 按钮开关
- 额定电流：5A/10A
- 电压：AC220V/AC380V/DC24V/DC220V
- 类型：
  - 平头按钮(自复位/自锁)
  - 蘑菇头按钮(急停)
  - 带灯按钮
  - 旋钮开关
  - 钥匙开关
- 颜色：红、绿、黄、蓝、白、黑

### TGLD系列 - 指示灯
- 额定电压：AC220V/AC380V/DC24V/DC220V
- 光源：LED(高亮度、长寿命)
- 颜色：红、绿、黄、蓝、白
- 尺寸：φ16mm、φ22mm

### TGLW系列 - 万能转换开关
- 额定电流：10A/20A/32A
- 电压：AC380V/AC220V
- 档位：2位、3位、多位
- 触点：多组转换触点

## 型号命名规则

### 按钮
```
TGLA-[类型] [颜色] [触点] [电压]
```
示例：
- `TGLA-11 绿色 1NO+1NC AC220V` - 平头按钮，绿色，1常开1常闭
- `TGLA-11D 红色 带灯 AC220V` - 带灯按钮，红色

### 指示灯
```
TGLD-[颜色] [电压] [尺寸]
```
示例：
- `TGLD-红色 AC220V φ22` - 红色指示灯，AC220V，22mm

## 按钮类型

| 类型 | 特点 | 应用 |
|------|------|------|
| 平头按钮 | 标准型，自复位 | 启动、停止 |
| 蘑菇头按钮 | 大按钮，紧急操作 | 急停按钮 |
| 带灯按钮 | 集成指示灯 | 状态显示+操作 |
| 旋钮开关 | 2位或3位保持 | 模式选择 |
| 钥匙开关 | 需钥匙操作 | 权限控制 |

## 指示灯颜色含义

| 颜色 | 含义 | 应用 |
|------|------|------|
| 红色 | 停止、故障、危险 | 急停指示、故障报警 |
| 绿色 | 启动、运行、正常 | 运行指示、电源指示 |
| 黄色 | 警告、注意 | 过载警告、异常状态 |
| 蓝色 | 强制、特殊 | 特殊功能指示 |
| 白色 | 一般指示 | 通用指示 |

## 选型要点

### 按钮选型
1. **功能** - 启动/停止/急停/选择
2. **触点** - 常开(NO)、常闭(NC)数量
3. **颜色** - 按功能选择(红停绿启)
4. **电压** - 与控制回路电压一致
5. **是否带灯** - 需要状态显示时选带灯按钮

### 指示灯选型
1. **颜色** - 按状态含义选择
2. **电压** - 与控制回路电压一致
3. **尺寸** - φ22mm(标准)或φ16mm(小型)
4. **光源** - LED(推荐，寿命长)

## 典型应用

1. **控制柜面板** - 启动/停止按钮，运行/故障指示灯
2. **配电箱** - 电源指示、开关状态指示
3. **机械设备** - 急停按钮、模式选择开关
4. **自动化系统** - 带灯按钮、状态指示灯

## 安装方式

- 面板开孔：φ22mm(标准)、φ16mm(小型)
- 固定方式：螺纹固定、卡扣固定
- 接线方式：螺钉端子、插片端子
