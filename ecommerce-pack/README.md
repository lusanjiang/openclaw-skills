# 🛒 电商运营技能包

专为电商卖家设计的AI技能包，提供产品图片生成、SEO优化文案、竞品分析和市场研究等全方位运营支持。

## 🚀 快速开始

```bash
# 产品图生成（即梦AI）
python3 jimeng_ai.py "白色背景 无线耳机"

# SEO文案
#电商 写标题 智能手表 主打健康监测

# 竞品分析
#搜索 蓝牙耳机 竞品 天猫

# 市场研究
深度研究 智能手表 电商市场规模
```

| ✍️ SEO Writer | 关键词优化、标题/文案生成 | ✅ |
| 🌐 网页搜索 & 分析 | 竞品监控、行业动态 | ✅ |

## 🎨 即梦AI配置

1. 访问 https://jimeng.jianying.com
2. 抖音账号登录
3. F12 → Application → Cookies → 复制 sessionid
4. `export JIMENG_SESSION_ID="你的sessionid"`

## 📊 特色功能

- **产品图生成**：即梦AI接入，白底图、场景图、Banner一键生成
- **SEO优化**：标题、详情页关键词智能布局
- **竞品监控**：价格、销量、策略全方位分析
- **市场研究**：行业报告、消费洞察、机会挖掘
