## 📝 SEO文案最佳实践

### 产品标题结构
```
[品牌] + [核心关键词] + [卖点1] + [卖点2] + [规格]

示例：
Apple AirPods Pro 主动降噪无线耳机 长续航 蓝牙5.0
```

### 详情页结构（AIDA模型）
1. **Attention** - 首屏吸引（大图+核心卖点）
2. **Interest** - 兴趣激发（场景图+功能介绍）
3. **Desire** - 欲望刺激（对比图+用户评价）
4. **Action** - 行动号召（促销信息+购买按钮）

### 关键词布局
- **标题**：核心关键词前置
- **详情页**：关键词自然分布，密度2-5%
- **属性**：填写完整，覆盖长尾词

## 🖼️ 图片生成技巧

### 提示词公式
```
[主体] + [场景] + [风格] + [光线] + [质量]

示例：
无线耳机，白色背景，产品摄影，柔和光线，8K高清
智能手表，户外跑步场景，运动风格，自然光，电商主图
```

### 电商常用场景
| 类型 | 提示词示例 |
|:---|:---|
| 主图 | `白色背景，产品居中，专业摄影，高清` |
| 场景图 | `家居环境，温馨氛围，自然光，生活化` |

## ❓ 常见问题

**Q: AI生成的图片可以直接商用吗？**
A: 建议查看具体AI模型的使用条款，一般生成内容可商用但需遵守平台规则。

**Q: SEO文案会被搜索引擎惩罚吗？**
A: 遵循白帽SEO原则，内容为王，关键词自然分布，不会被惩罚。

**Q: 竞品数据准确吗？**
A: 搜索数据基于公开信息，价格和销量可能有延迟，建议作为参考而非绝对依据。
