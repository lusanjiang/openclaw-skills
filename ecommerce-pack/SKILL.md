## 🧩 包含技能 4个

### 1. 🖼️ AI 图片生成（即梦AI接入）
一句话生成电商产品图、主图、详情页配图

**🎯 已接入：即梦AI (Jimeng)**
- **平台**：字节跳动旗下AI创作平台
- **模型**：即梦4.0（中文优化）
- **费用**：新用户免费试用，每日送积分
- **网址**：https://jimeng.jianying.com

**能力**：
- 文生图：产品主图、场景图、白底图
- 多风格：简约/高端/科技感/温馨/活力
- 自定义：尺寸、背景、风格参数可调
- 批量生成：支持一次生成多张

**使用示例**：
- `#电商 生成产品图 白色背景 蓝牙耳机`
- `#电商 生成产品图 高端风格 智能手表 黑色背景`
- `python3 jimeng_ai.py "白色背景 蓝牙耳机"`

**配置方法**：
1. 访问 https://jimeng.jianying.com 用抖音账号登录
2. F12打开开发者工具 → Application → Cookies
3. 复制 `sessionid` 值
4. 设置环境变量：`export JIMENG_SESSION_ID="你的sessionid"`

**状态**：✅ 已就绪（需配置Session ID）

### 即梦AI配置步骤

**第一步：注册登录**
1. 访问 https://jimeng.jianying.com
2. 使用抖音账号扫码登录

**第二步：获取Session ID**
```
1. 登录后按 F12 打开开发者工具
2. 切换到 Application（应用）标签
3. 左侧找到 Storage → Cookies → jimeng.jianying.com
4. 找到名为 sessionid 的项，复制其值
```

**第三步：配置环境变量**
```bash
# 临时配置（当前终端有效）
export JIMENG_SESSION_ID="你的sessionid"

# 永久配置（添加到 ~/.bashrc）
echo 'export JIMENG_SESSION_ID="你的sessionid"' >> ~/.bashrc
source ~/.bashrc
```

**第四步：测试使用**
```bash
cd /root/.openclaw/skills/ecommerce-pack
python3 jimeng_ai.py "白色背景 蓝牙耳机"
```
