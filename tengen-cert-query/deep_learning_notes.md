# 天正电气证书查询 - 深度学习笔记
生成时间: 2026-03-06 23:45

## 一、证书命名规律总结

### 1. 证书类型标识
| 类型 | 命名前缀 | 示例 |
|:---|:---|:---|
| CCC证书 | `CCC证书-` | CCC证书-TGB1N-63 |
| CCC报告 | `CCC报告-` | CCC报告-TGM1N-400 |
| CE证书 | `CE证书-` | CE证书-TGB1N-125 |
| CE报告 | `CE报告-` | CE报告-TGB1N-125 |
| CB证书 | `CB证书-` | CB证书-TGB1N-63 |
| CB报告 | `CB报告-` | CB报告-TGB1N-63 |
| CQC报告 | `CQC报告-` | CQC报告-TGC1-2511 |

### 2. 多型号合并规律

#### 微断系列（TGB1N/TGBG/TGBH/TGBK/TGBD）
- 一份证书常覆盖5-6个系列
- 例：`CCC证书-TGB1N-63、TGBH-63、TGBG-63、TGBG-63H、TGBG-63A、TGBG-63HA`

#### 塑壳系列（TGM1N/TGMH/TGMK/TGMG/TGMD）
- 一份证书覆盖多个系列+多个电流档位
- 例：`CCC证书-TGM1N-400,TGMH-400,TGMK-400,TGMG-400,TGMD-400,TGM1N-630,TGMH-630...`

#### 框架系列（TGW1N/TGWG/TGWK）
- 一份证书覆盖同电流档位的不同派生型号
- 例：`CCC证书-TGW1N-3200、TGW1N-3200H、TGW1N-3200L`

#### 接触器系列（TGC1/TGCG/TGCJH/TGCK/TGCA）
- 一份证书覆盖同电流档位的不同系列
- 例：`CCC证书-TGC1-0911,TGC1-1211,TGC1-1811,TGCG-0911...`

### 3. 电流分档规律

#### 微断（TGB1N系列）
- 常见电流：40A、63A、125A
- 派生型号后缀：A、H、M、N、S、DC

#### 塑壳（TGM1N系列）
- 常见电流：63A、125A、400A、630A、800A、1250A、1600A
- 派生型号后缀：L、M、H、R

#### 框架（TGW1N系列）
- 常见电流：2000A、2500A、3200A、4000A、6300A
- 派生型号后缀：H、L、C

#### 接触器（TGC1系列）
- 常见电流：06、09、12、18、25、32、38、40、50、65、500、630、760
- 派生型号后缀：N（某种派生）

### 4. 派生后缀含义总结

| 后缀 | 含义 | 适用系列 |
|:---|:---|:---|
| M | 标准型 | TGM1N/TGB1N |
| H | 高分断型 | 全系列 |
| L | 低分断型 | 全系列 |
| C | 某种派生（框架专用） | TGW1N |
| R | 某种派生（塑壳专用） | TGM1N |
| S | 特殊型 | TGB1N |
| N | 某种派生（微断/接触器） | TGB1N/TGC1 |
| A | 某种派生 | TGB1N |
| DC | 直流专用 | 全系列 |
| LE | 电子式漏电 | TGB1NLE |
| LA | 电子式漏电A型 | TGB1NLA |
| L | 漏电 | TGB1NL |

### 5. 证书编号规律

官网证书文件路径格式：
```
https://www.tengen.com/public/uploads/files/YYYYMMDD/[32位哈希].pdf
```

- 日期格式：年月日（如20240701）
- 哈希值：32位小写十六进制字符串

## 二、查询优化建议

### 1. 微断查询技巧
- 查询TGB1N-63时，系统会返回包含TGB1N-63的证书（可能同时包含TGBH/TGBG等）
- 这是正常的，因为天正习惯将多个系列合并于一份证书

### 2. 塑壳查询技巧
- 查询TGM1N-400时，可能会返回同时包含400A和630A的证书
- 这是正常的，因为塑壳常将相邻电流档位合并

### 3. 框架查询技巧
- 查询TGW1N-3200时，可能会返回同时包含2500A和3200A的证书
- 这是正常的，因为框架常将相邻电流档位合并

### 4. 接触器查询技巧
- 接触器系列证书命名格式较复杂，常包含多个电流值
- 例：`CCC证书-TGC1-0911,TGC1-1211,TGC1-1811...`

## 三、常见问题处理

### Q1: 为什么查TGB1N-63会返回包含TGBH-63的证书？
A: 天正将多个系列（TGB1N/TGBH/TGBG/TGBK/TGBD）合并于一份证书，这是正常情况。

### Q2: 为什么查TGM1N-400会返回包含630A的证书？
A: 天正将相邻电流档位（400A+630A）合并于一份证书，这是正常情况。

### Q3: 如何区分证书和报告？
A: 看命名前缀：
- `CCC证书-`、`CE证书-`、`CB证书-` → 证书
- `CCC报告-`、`CE报告-`、`CB报告-`、`CQC报告-` → 报告

### Q4: 为什么有些系列查不到证书？
A: 可能原因：
1. 该系列合并于其他系列证书中（如TGBK合并于TGB1N）
2. 该系列为新产品，证书尚未上传
3. 该系列证书命名方式不同

## 七、学习更新（23:47）

### TUV证书分布
- TUV证书存在于官网，独立命名：`TUV证书-型号`
- 主要覆盖：TGB3-40(新)、TGC1-25/32/38、TGC1-80/95
- 并非所有产品都有TUV证书

### 型式试验报告分布
- 型式试验报告主要存在于**高压产品**
- 文件日期较早（2019年左右）
- 命名格式：`型号 型式试验报告`
- 例：GN30-12 3150-31.5 型式试验报告

### ODM证书
- 部分证书标注ODM，如：`CCC证书-TGM1N-1600 ODM`
- ODM证书与常规证书格式相同

### 电流参数标注规律
- 塑壳TGM1N-800证书包含：L、M、H后缀（低/标准/高分断）
- 证书中电流参数标注方式：直接跟在后缀中
