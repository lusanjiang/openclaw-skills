## 二、证书类型详解

### 2.1 CCC（中国强制认证）
- **适用范围**: 所有在中国销售的产品
- **存在形式**: 证书 + 报告
- **数量**: 最多，覆盖全系列
- **查询建议**: 默认查询类型

### 2.2 CE（欧盟认证）
- **适用范围**: 出口欧洲的产品
- **存在形式**: 证书 + 报告
- **主要产品**: TGM1N, TGB1N, TGBG等
- **数量**: 50+个

### 2.3 CB（国际电工委员会认证）
- **适用范围**: 微断系列
- **存在形式**: 证书 + 报告
- **主要产品**: TGB1N, TGBG
- **数量**: 50+个

### 2.4 TUV（德国莱茵认证）
- **适用范围**: 特定产品
- **存在形式**: 独立证书
- **主要产品**: TGB3, TGC1
- **数量**: 50+个（但独立TUV证书较少）

### 2.5 CQC（中国质量认证）
- **适用范围**: 接触器等
- **存在形式**: 报告为主
- **主要产品**: TGC1
- **数量**: 50+个

### 2.6 其他认证
| 认证 | 存在性 | 数量 | 说明 |
|:---|:---:|:---:|:---|
| UL | ✅ | 7个 | 极少，TGM5DC等 |
| RoHS | ✅ | 24个 | 报告形式 |
| ISO | ✅ | 14个 | 公司体系认证 |
| 型式试验 | ✅ | 16个 | 高压产品 |

## 四、证书命名规律

### 4.1 微断系列
```
CCC证书-TGB1N-63、TGBH-63、TGBG-63、TGBG-63H、TGBG-63A、TGBG-63HA
↑类型   ↑主型号 ↑系列2   ↑系列3   ↑派生H  ↑派生A  ↑派生HA
```

### 4.2 塑壳系列
```
CCC证书-TGM1N-400,TGMH-400,TGMK-400,TGMG-400,TGMD-400,TGM1N-630,TGMH-630...
↑类型   ↑主型号-400A ↑系列2-400A ... ↑主型号-630A ↑系列2-630A ...
```

### 4.3 框架系列
```
CCC证书-TGW1N-2000、TGW1N-2000H、TGW1N-2000C、TGW1N-2000L
↑类型   ↑主型号    ↑派生H      ↑派生C      ↑派生L
```

### 4.4 双电源系列
```
CCC证书-TGQ1N-160L、TGQ1N-160M
↑类型   ↑主型号-电流-低分断 ↑主型号-电流-标准分断

CCC证书-TGQ1N-250L3、4，TGQ1N-250M3、4
↑类型   ↑主型号-电流-低分断-3/4极 ↑主型号-电流-标准分断-3/4极
```

## 五、常见问题FAQ

### Q1: 为什么查询TGB1N-32提示"未找到"？
**A:** TGB1N小电流档位（10-32A）可能合并于TGB1N-40证书中。建议查询TGB1N-40。

### Q2: 为什么查询TGQ1N-160提示"未找到"？
**A:** TGQ1N双电源命名特殊，需要添加分断类型后缀。正确查询方式："TGQ1N-160L"或"TGQ1N-160M"。

### Q3: 如何区分证书和报告？
**A:** 看命名前缀：
- `CCC证书-`、`CE证书-`、`CB证书-` → 证书
- `CCC报告-`、`CE报告-`、`CB报告-`、`CQC报告-` → 报告

### Q4: 为什么查TGM1N-400会返回包含630A的证书？
**A:** 天正将相邻电流档位（400A+630A）合并于一份证书，这是正常情况。

### Q5: 为什么有些系列查不到证书？
**A:** 可能原因：
1. 该系列合并于其他系列证书中（如TGBK合并于TGB1N）
2. 该系列命名方式特殊（如TGQ1N需要L/M后缀）
3. 该系列为新产品，证书尚未上传

### Q6: UL证书多吗？
**A:** UL认证不是天正主要方向，官网仅找到7个UL证书，主要是TGM5DC等特殊型号。

### Q7: RoHS证书怎么查？
**A:** RoHS主要以报告形式存在，查询时应该说"RoHS报告"而非"RoHS证书"。

## 七、高级查询技巧

### 7.1 模糊查询
如果精确型号查不到，可以尝试：
```
#查询证书 TGB1N-40（小电流档位可能合并于此）
#查询证书 TGB1N-63（小电流可能合并，大电流可能合并）
```

### 7.2 系列合并查询
查询派生系列时，可以尝试主系列：
```
#查询证书 TGB1N-63（可能包含TGBH/TGBG/TGBK/TGBD）
#查询证书 TGM1N-400（可能包含TGMH/TGMK/TGMG/TGMD）
```

### 7.3 多认证类型查询
同一产品可能有多重认证：
```
#查询证书 TGB1N-63的CCC证书
#查询证书 TGB1N-63的CE证书
#查询证书 TGB1N-63的CB证书
```
