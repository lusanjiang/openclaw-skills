
## 证书类型说明

| 证书类型 | 适用产品 | 用途 |
|:---|:---|:---|
| **CCC** | 全系列 | 中国强制认证，国内销售必备 |

| **CQC** | TGC1接触器 | 中国质量认证（多为报告） |

## 常见问题

### Q1: 为什么查询TGB1N-63会返回包含TGBH-63的证书？
**A:** 天正将多个系列（TGB1N/TGBH/TGBG/TGBK/TGBD）合并于一份证书，这是正常情况。证书中的TGB1N-63部分符合您的需求。

### Q2: 为什么查询TGM1N-400会返回包含630A的证书？
**A:** 天正将相邻电流档位（400A+630A）合并于一份证书，这是正常情况。

### Q3: 如何区分证书和报告？
**A:** 看命名前缀：
- `CCC证书-`、`CE证书-`、`CB证书-` → 证书
- `CCC报告-`、`CE报告-`、`CB报告-`、`CQC报告-` → 报告

### Q4: 为什么有些系列查不到证书？
**A:** 可能原因：
1. 该系列合并于其他系列证书中（如TGBK合并于TGB1N）
2. 该系列为新产品，证书尚未上传
3. 该系列证书命名方式不同

### Q5: 如何获取更精确的查询结果？
**A:** 
1. 使用完整型号+完整证书类型，如："TGB1N-63的CCC证书"
2. 系统会自动过滤其他型号变体（如TGB1N-63DC）
3. 系统会自动按文档类型筛选（证书vs报告）
