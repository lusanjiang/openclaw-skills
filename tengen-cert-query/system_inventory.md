## 三、证书命名规律

### 1. CCC证书命名
```
CCC证书-主型号、派生型号1、派生型号2...
例: CCC证书-TGB1N-63、TGBH-63、TGBG-63...
```

### 2. CE证书命名
```
CE证书-型号1,型号2,型号3...
例: CE证书-TGM1N-400,TGMH-400,TGMK-400...
```

### 3. CQC报告命名
```
CQC报告-型号1,型号2...
例: CQC报告-TGC1-2511,TGCG-2511...
```

### 4. CB证书命名
```
CB证书-型号1,型号2...
例: CB证书-TGB1N-63A,TGB1N-63...
```
