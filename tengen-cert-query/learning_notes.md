# 天正电气证书查询 - 学习总结

## 证书类型分布

### 1. CCC证书（最常见）
- **覆盖产品**: 几乎所有产品都有CCC证书
- **命名格式**: `CCC证书-型号1,型号2,型号3...`
- **覆盖规律**: 
  - 框架：按电流分档（4000A、3200A、2500A等）
  - 塑壳：多型号合并（TGM1N/TGMH/TGMK/TGMG/TGMD等）
  - 微断：系列合并（TGB1N/TGBH/TGBG/TGBK等）
  - 双电源：按电流分档（800A、320A、250A、160A等）

### 2. CE证书
- **覆盖产品**: 出口产品（TGM1N、TGB1N等）
- **命名格式**: `CE证书-型号1,型号2...`
- **特点**: 多型号合并，常与TUV一起

### 3. CQC报告
- **覆盖产品**: 接触器TGC1系列
- **命名格式**: `CQC报告-型号1,型号2...`
- **特点**: 有CQC报告，没有CQC证书

### 4. CB证书
- **覆盖产品**: 微断TGB1N系列
- **命名格式**: `CB证书-型号1,型号2...`

### 5. TUV证书
- **发现**: 常以"TUV，CE证书"形式出现，少有单独TUV证书

### 6. 其他证书（暂未发现）
- UL证书：未找到
- RoHS证书：未找到  
- 型式试验报告：命名可能不同

## 查询技巧

1. **多型号合并证书**: 一份证书常覆盖多个相近型号
   - 例：CCC证书-TGB1N-63包含TGBH/TGBG/TGBK等
   
2. **派生型号命名**:
   - M = 标准型
   - H = 高分断
   - L = 低分断
   - S = 特殊型
   - A = 某类附件
   - DC = 直流专用

3. **按电流分档**: 框架/塑壳常按额定电流分多份证书

4. **证书与报告区别**:
   - 证书 = 认证证书（CCC/CE/CB）
   - 报告 = 测试报告（CQC报告/型式试验）
