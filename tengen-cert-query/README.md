## 🎓 学习成果

### 证书命名规律
- **微断**: 一份证书覆盖5-6个系列（TGB1N/TGBH/TGBG/TGBK/TGBD）
- **塑壳**: 一份证书覆盖多系列+多电流档位（400A+630A合并）
- **框架**: 一份证书覆盖同电流档位的不同派生型号
- **双电源**: 必须包含分断类型后缀（L=低分断, M=标准分断）
- **接触器**: 全电流档位独立证书

### 派生后缀含义
| 后缀 | 含义 |
|:---|:---|
| M | 标准型 |
| H | 高分断型 |
| L | 低分断型 |
| C | 某种派生（框架）|
| R | 某种派生（塑壳）|
| S | 特殊型 |
| DC | 直流专用 |
| LE | 电子式漏电 |

## ⚠️ 注意事项

1. **TGB1N小电流**: 10-32A可能合并于40A证书
2. **TGQ1N双电源**: 必须加L/M后缀（如TGQ1N-160L）
3. **TGW1N-5000**: 较少见，可能合并于6300A
4. **UL证书**: 极少（仅7个），不要期望过高
5. **RoHS**: 报告形式，查询时说"RoHS报告"
