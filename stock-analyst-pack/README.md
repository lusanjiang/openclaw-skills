专为股票投资者设计的AI技能包，提供实时行情查询、深度研报分析、量化策略开发和智能投资决策支持。

- **量化信号监控**：RSI/MACD/布林带自动生成买卖信号
- **实时推送**：交易时段内即时推送，非交易时段自动静默
- **持仓追踪**：自动跟踪涨跌幅，异常波动预警
