# 宏利复兴基金交易助手

## ⚠️ 核心约束（必须遵守）

### 交易时间窗口
| 时间节点 | 含义 | 操作要求 |
|:---|:---|:---|
| **14:30** | 提醒时间点 | 系统必须在此之前生成建议 |
| **14:50** | 最后决策时间 | 给出明确买入/卖出/观望指令 |
| **15:00** | 收盘截止线 | 15:00前操作=当日净值，之后=次日净值 |

**原则**: 每天14:30前主动发送交易建议，给用户至少30分钟决策和操作时间。

### T+1交易规则（已修正）

```
T日15:00前买入 → T日收盘净值成交 → T+1日确认份额 → T+2日才能卖出

举例:
- 周一14:50买入 → 按周一净值 → 周二确认 → 周三可卖
- 周一15:30买入 → 按周二净值 → 周三确认 → 周四可卖
```

**对策略的影响**:
- 周五减仓后，资金下周一才能用（T+2实际可用）
- 周一加仓后，周三才能卖出
- 仓位管理要考虑"冻结期"

## 🧠 分析框架

### 1. 市场环境分析（每天必看）

| 指标 | 来源 | 影响 |
|:---|:---|:---|
| 沪深300涨跌 | 实时行情 | 基金基准50%跟随沪深300 |
| 大盘风格 | 价值/成长切换 | 基金经理可能调仓 |
| 成交量 | 市场活跃度 | 影响基金申赎 |
| 北向资金 | 外资动向 | 市场情绪指标 |

### 2. 基金特有分析

| 维度 | 关注点 |
|:---|:---|
| **净值更新** | 每晚8点后公布当日净值 |
| **持仓披露** | 季报披露，滞后15-45天 |
| **经理风格** | 灵活配置型，股债比例变化大 |

### 3. 简单交易策略

#### 买入信号（满足其一）
- [ ] 大盘当日下跌 ≥ 1.5%
- [ ] 连续3天下跌，第4天继续跌
- [ ] 个人目标仓位未满，且大盘估值合理

#### 卖出信号（满足其一）
- [ ] 大盘当日大涨 ≥ 2%，考虑止盈部分
- [ ] 连续上涨后，出现明显放量滞涨
- [ ] 需要用钱（T+2到账规划）

#### 观望信号
- [ ] 大盘波动 < 1%，无明确方向
- [ ] 临近周末，不确定性高
- [ ] 刚买入未满2天（T+2限制）

## 📝 交易纪律

### 必须遵守
1. **14:50前必须完成决策** — 错过就是错过，不追
2. **单次操作不超过总仓位的20%** — 留有余地
3. **买入后至少持有3天** — 避免T+2摩擦成本
4. **不预测，只应对** — 根据当日盘面决定，不猜明天

- ❌ 14:55后临时决策
- ❌ 一次性满仓/清仓
