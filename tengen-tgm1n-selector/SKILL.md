## 🏗️ 工作流架构 (v2.0 新特性)

学习自 Coze 工作流设计理念，选型流程可视化编排：

```
用户输入 → 解析参数 → 验证电流 → 分断建议 → 生成型号 → EAP3查询 → 输出结果
    │           │           │           │           │           │
    ▼           ▼           ▼           ▼           ▼           ▼
  #tgm1n    提取壳架    检查规格    L/M/H/R    TGM1N-    价格货期   格式化
            电流极数    是否在列    分断能力   125M/3300            报告
```

### 节点类型
| 节点 | 功能 | 说明 |
|:---|:---|:---|
| `parse_input` | 解析用户输入 | 提取壳架、电流、极数、附件、电压 |
| `validate_current` | 验证电流规格 | 检查电流是否在壳架规格范围内 |
| `recommend_breaking` | 分断能力建议 | 推荐L/M/H/R型及分断能力 |
| `generate_model` | 生成完整型号 | 按命名规则生成标准型号 |
| `query_eap3` | EAP3价格查询 | 查询价格和货期 |
| `format_output` | 格式化输出 | 生成可读报告 |

## 选型流程

### 标准选型工作流

| 步骤 | 节点 | 说明 |
|------|------|------|
| **1** | `parse_input` | 解析用户输入，提取壳架、电流、极数、附件、电压 |
| **2** | `validate_current` | 验证电流是否在壳架规格范围内 |
| **3** | `recommend_breaking` | 推荐分断能力代码（L/M/H/R）|
| **4** | `generate_model` | 生成完整型号（含附件代码）|
| **5** | `query_eap3` | 调用EAP3查询价格和货期 |
| **6** | `format_output` | 格式化输出选型报告 |

## 型号命名规则

```
TGM1N-[壳架][分段]/[极数][脱扣][附件] [电流] [电压](祥云3.0)
```

### 分段代码
| 代码 | 分断能力 | 适用场景 |
|:---|:---|:---|
| L | 25kA | 低要求场合 |
| M | 35-50kA | **默认推荐** |
| H | 50-70kA | 高分断要求 |
| R | 70-100kA | 极高要求（限流型）|

### 附件代码
| 代码 | 附件组合 | 代码 | 附件组合 |
|:---|:---|:---|:---|
| 00 | 无附件 | 40 | 分励+辅助 |
| 10 | 分励脱扣器 | 48 | 分励+辅助+报警 |
| 18 | 分励+报警 | 50 | 分励+欠压 |
| 20 | 辅助触头 | 60 | 二组辅助 |
| 28 | 辅助+报警 | 70 | 辅助+欠压 |
| 30 | 欠压脱扣器 | 78 | 欠压+辅助+报警 |

### 特殊后缀
| 后缀 | 含义 |
|:---|:---|
| 2 | 电动机保护（附件代码后加）|
| Ⅲ | 过载报警不脱扣 |
| T | 透明盖 |
| B | 板后接线 |
| C | 插入式 |

## 注意事项

1. 含分励/欠压的型号必须标注电压（AC220V/AC380V/DC24V）
2. 电动机保护需在附件代码后加'2'
3. 4P默认为B型（N极参与合分），3N为A/D型（N极始终接通）
4. 630大体积/800/1250/1600壳架暂无透明盖产品
5. 2P暂无透明盖产品
