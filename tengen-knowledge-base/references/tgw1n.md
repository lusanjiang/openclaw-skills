# TGW1N系列万能式断路器

## 产品概述

TGW1N系列万能式断路器适用于交流50Hz，额定电压AC380V~AC690V，额定电流200A~6300A的配电网络中，用来分配电能和保护线路及电源设备免受过载、短路、欠电压、单相接地或剩余电流等故障的危害。

## 技术参数

| 参数 | 规格 |
|------|------|
| 额定电压 | AC380V~AC690V |
| 额定频率 | 50Hz |
| 额定电流 | 200A~6300A |
| 分断能力 | 50kA/65kA/80kA/100kA |
| 极数 | 3P, 4P |

## 壳架等级

|------|-------------|----------|

## 控制器类型

### 祥云3.0控制器（M型）
- 标准配置
- 支持基本保护功能
- 可选配通信功能

### 3H型控制器
- 高级配置
- 支持检有压合闸功能
- 支持更多附件组合

## 附件配置

| 附件名称 | 代码 | 说明 |
|----------|------|------|
| 欠压脱扣器 | Q | 失压自动跳闸，**常规不延时** |
| 欠压延时脱扣器 | Q+延时 | 失压保护，带延时功能（需明确备注延时时间）|
| 分励脱扣器 | F | 远程控制跳闸 |
| 辅助触头 | AX | 状态信号输出 |
| 报警触头 | AL | 故障报警 |
| 漏电保护 | V | 外接零序互感器 |
| 检有压模块 | S | 电压检测合闸 |

### 选型规则约定

**欠压脱扣器约定：**
- 用户说 **"欠压"** → 默认常规不延时
- 用户说 **"欠压延时Xs"** → 带延时功能（如欠压延时1s、欠压延时3s、欠压延时5s、欠压延时7s）
- 延时时间必须明确备注，否则按常规不延时处理

## 型号命名规则

```
TGW1N-[壳架]/[极数] [电流] [安装] [控制电压] [附件]
```

示例：
- `TGW1N-2000/3P 2000A 抽屉水平 AC220V（祥云3.0）`
- `TGW1N-1000/4P 800A 固定式 AC220V 带漏电（祥云3.0）`

## 认证标准

- 3C认证

- CB认证
- 符合GB/T14048.2、IEC/EN60947-2标准

## 选型注意事项

1. **分断能力选择** - 根据系统短路电流确定
2. **控制电压匹配** - 与二次回路电压一致
3. **附件组合** - 检有压功能有两种实现方式，价格差异大
4. **安装空间** - 抽屉式需要更多柜体深度
