# TGWD系列万能式断路器

## 产品概述

TGWD系列万能式断路器是天正电气的高端产品线，采用2H/3H/3M型智能控制器，适用于交流50Hz，额定电压AC380V~AC690V，额定电流200A~6300A的配电网络。

## 技术参数

| 参数 | 规格 |
|------|------|
| 额定电压 | AC380V~AC690V |
| 额定频率 | 50Hz |
| 额定电流 | 200A~6300A |
| 极数 | 3P, 4P |

## 壳架等级

| 壳架 | 额定电流范围 |
|------|-------------|
| TGWD-1000 | 200A~1000A |
| TGWD-2000 | 800A~2000A |
| TGWD-3200 | 2000A~3200A |
| TGWD-4000 | 2500A~4000A |
| TGWD-6300 | 4000A~6300A |

## 控制器类型

### 2H型控制器
- 基础智能型
- 支持基本保护和测量
- 适用于标准配电场合

### 3H型控制器
- 高级智能型
- 支持四遥功能（遥控、遥信、遥测、遥调）
- 支持更多附件组合

### 3M型控制器
- 经济型
- 基本保护功能
- 性价比高

## 安装方式

- **固定水平** - 固定安装，水平接线
- **抽屉水平** - 抽屉式安装，水平接线

## 附件配置

| 附件名称 | 说明 |
|----------|------|
| 欠压延时脱扣器 | 带延时功能的欠压保护 |
| 分励脱扣器 | 远程分闸控制 |
| 辅助触头 | 四开四闭、六组转换等 |
| 四遥功能 | 遥控、遥信、遥测、遥调 |
| 漏电保护 | 外接零序互感器 |
| 接线端子 | 55回路、长排/短排 |

## 型号命名规则

```
TGWD-[壳架]/[极数] [电流] [安装] [控制器] [控制电压] [附件]
```

示例：
- `TGWD-2000/3P 1600A 抽屉水平 3M型 DC220V`
- `TGWD-3200/3P 2500A 抽屉水平 3H型 AC220V 带四遥功能(长排)`

## 特殊标识说明

| 标识 | 含义 |
|------|------|
| (装锁) | 带锁功能，可防止误操作 |
| (U) | 特殊电压版本 |
| (长排) | 长排接线方式 |
| (短排) | 短排接线方式 |
| (FJJP) | 福建精品版本 |
| Ir1=0.8In | 长延时整定值为0.8倍额定电流 |
| 欠压延时Xs | 欠压脱扣延时X秒 |

## 与TGW1N的区别

| 对比项 | TGWD | TGW1N |
|--------|------|-------|
| 控制器 | 2H/3H/3M型 | 祥云3.0/3H型 |
| 安装方式 | 固定水平、抽屉水平 | 固定式、抽屉水平、抽屉垂直 |
| 接线端子 | 55回路 | 56回路 |
| 特色功能 | 四遥功能 | 检有压合闸 |
| 价格定位 | 中高端 | 中端 |

## 典型应用场景

1. **配电房进线柜** - 大电流、高可靠性要求
2. **母联柜** - 双电源自动切换
3. **重要负荷出线** - 需要远程监控的回路
4. **数据中心** - 需要四遥功能的场合

## 选型注意事项

1. **控制器选择** - 3M经济、2H标准、3H高端
2. **接线方式** - 确认长排/短排需求
3. **四遥功能** - 需要远程监控时选3H型
4. **欠压延时** - 根据系统要求选择延时时间
