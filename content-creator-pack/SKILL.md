# 🎨 内容创作者技能包

## 技能清单

### 1. 💚 微信公众号
- **功能**：公众号全流程运营
- **能力**：
  - 选题策划
  - 文章写作
  - 自动排版
  - 配图生成
  - 发布到草稿箱
- **特点**：Markdown一键转公众号格式
- **使用示例**：
  - "写一篇关于AI的公众号文章"
  - "把这篇报告改成公众号格式"
- **状态**：⚠️ 需要微信公众号权限

### 2. ✍️ 博客写手
- **功能**：个人博客全流程写作
- **能力**：
  - 长文写作
  - SEO优化
  - 系列策划
  - 自动配图
  - 发布到博客平台
- **特点**：持续产出高质量内容
- **使用示例**：
  - "写一篇技术博客"
  - "优化这篇博客的SEO"
- **状态**：⚠️ 需要博客平台接入

### 3. 🖼️ AI 图片生成
- **功能**：一句话生成图片
- **模型**：10+ AI模型可选
- **能力**：
  - 文生图
  - 图生图
  - 风格迁移
  - 局部重绘
  - 产品图/艺术画
- **特点**：秒级交付，多种风格
- **使用示例**：
  - "生成一张科技风格封面图"
  - "把这张图转换成油画风格"
- **状态**：⚠️ 需要AI绘图API

### 4. 🧑 AI 文本人性化
- **功能**：去除AI痕迹，让文字更自然
- **能力**：
  - 6种风格变换
  - 16种检测模式
  - 降低AI检测率
  - 增强文本可读性
- **特点**：让AI生成内容更像人类写作
- **使用示例**：
  - "把这段话改得更自然"
  - "去除这段文字的AI痕迹"
- **状态**：⚠️ 需要人性化API

## 快速使用

```
#创作者          - 查看技能包状态
#公众号 写文章    - 微信公众号
#博客 技术文章    - 博客写手
#画图 科技封面    - AI图片生成
#人性化 这段文字  - AI文本人性化
```

## 配置要求

| 技能 | 配置需求 |
|:---|:---|
| 微信公众号 | 微信公众号账号/权限 |
| 博客写手 | 博客平台API接入 |
| AI图片生成 | AI绘图API（DALL-E/Midjourney等） |
| AI文本人性化 | 文本人性化API |

## 状态汇总

- ⚠️ 全部技能需要外部API配置
- 当前暂无内置实现

## 建议

这些技能需要接入外部服务API：
1. **微信公众号** - 需要微信开放平台认证
2. **博客写手** - 可对接Hexo/WordPress等平台
3. **AI图片生成** - 可使用DALL-E 3、Midjourney API
4. **AI文本人性化** - 可使用GPT-4、Claude等模型优化
