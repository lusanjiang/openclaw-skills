# @write 快捷写入工具

**版本**: v1.2  
**更新**: 2026-03-04  
**特性**: 具备自我校对机制 + 自我进化能力

## 快速调用

| 标签 | 用途 | 示例 |
|------|------|------|
| `#write` | 快速写入飞书文档 | #write 文档标题 内容... |

## 使用方法

### 创建新文档
```
#write 文档标题
这里是文档内容...
可以有多行
```

### 追加到现有文档
```
#write 文档标题 append
这是追加的内容...
```

### 指定文档token
```
#write doc_token:xxx 文档标题
内容...
```

## 参数说明

| 参数 | 说明 | 示例 |
|------|------|------|
| `文档标题` | 飞书文档标题 | TGM1N选型规则 |
| `append` | 追加模式（可选）| append |
| `doc_token:xxx` | 指定文档token（可选）| doc_token:Di4SdHddYovDz7xxstLcFa6Rngb |

## 示例

### 创建TGM1N文档
```
#write TGM1N塑壳断路器选型规则
# 天正电气 TGM1N 选型规则

## 一、型号命名规则
TGM1N-[壳架][分段]/[附件] [电流] [电压](祥云3.0)

## 二、壳架与分段能力
...
```

### 追加更新日志
```
#write TGM1N塑壳断路器选型规则 append

## 更新日志
### 2026-03-04
- 添加完整型号输出规范
```

## 技术实现

使用 `feishu_doc` 工具的 `create` 或 `append` action：

```python
# 创建文档
feishu_doc(action="create", title="标题", content="内容")

# 追加内容
feishu_doc(action="append", doc_token="xxx", content="追加内容")
```

## ✅ 自我校对机制（新增）

### 校对流程
```
写入内容 → 立即读取验证 → 检查内容完整性 → 确认成功/重试
```

### 校对检查项
| 检查项 | 标准 | 失败处理 |
|--------|------|----------|
| **内容非空** | block_count > 1 | 重试写入 |
| **标题匹配** | 文档标题与预期一致 | 报错提示 |
| **内容匹配** | 关键内容已写入 | 重试或报错 |
| **格式正确** | 无乱码、格式正常 | 重试写入 |

### 校对失败处理
1. **第一次失败** → 自动重试写入
2. **第二次失败** → 使用备用方式（分段写入）
3. **第三次失败** → 向用户报告失败原因，不交付空白文档

### 成功交付标准
✅ 文档内容完整可读  
✅ 关键章节已写入  
✅ 格式正确无乱码  

❌ **禁止交付空白文档**
