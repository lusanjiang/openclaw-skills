# Copywriting 文案写作技能

## 功能

- **理清结构**：帮助梳理文案逻辑框架
- **优化表达**：改进语言表达方式
- **写作反馈**：提供具体的改进建议
- **学习技巧**：传授文案写作方法论

## 使用方法

1. **输入文案**：提供你需要优化或学习的文案
2. **说明需求**：告诉我你想达到什么效果
3. **获得反馈**：我会分析并给出具体建议
4. **学习框架**：参考 frameworks.md 中的写作模型

## 适用场景

- 产品文案优化
- 营销文案撰写
- 内容结构梳理
- 表达技巧提升

## 写作框架

查看 frameworks.md 学习：
- AIDA 模型（注意-兴趣-欲望-行动）
- PAS 模型（问题-激化-解决）
- 4U 原则（紧迫-独特-具体-有用）
- 写作检查清单

## 原则

- 不是一键生成的套路货
- 注重逻辑和表达
- 提供可复用的方法论
- 通过练习提升写作能力
