# 什么值得买监控 Skill

## 快速使用

使用标签 `#我想买` 快速启动监控：

```
#我想买 燃气热水器
#我想买 iPhone 15
#我想买 机械键盘
```

## 功能

- 实时监控什么值得买优惠
- 自动筛选：评论>10条，值>80%，未过期
- 按热度排序，优先展示高性价比商品
- 支持任意关键词搜索

## 输出格式

```
🔥 找到 X 条有效商品：

| 排名 | 商品名称 | 价格 | 评论 | 值率 | 链接 |
|:---|:---|:---|:---:|:---:|:---|
| 1 | 商品A | ¥XXX | XX条 | XX% | [查看](链接) |
| 2 | 商品B | ¥XXX | XX条 | XX% | [查看](链接) |
```

## 筛选规则

| 条件 | 说明 |
|:---|:---|
| 评论数 | >10条（需超过10条评论）|
| 值率 | >85%（需超过85%值率）|
| 状态 | 未过期 |
| 排序 | 按热度/评论数 |

## 输出格式

```
🔥 找到 X 条有效商品：

1. 商品名称
   💰 价格
   💬 XX条评论
   📊 XX%值率
   🔗 链接
```

## 技术实现

- 调用什么值得买官方API
- 数据实时获取，非缓存
- 自动过滤过期和无效商品
